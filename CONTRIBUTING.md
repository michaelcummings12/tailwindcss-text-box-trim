# Contributing

Thank you for considering contributing to `tailwindcss-text-box-trim`!

## Development

1.  Clone the repo.
2.  Install dependencies: `npm install`.
3.  Make your changes in `index.css`.
4.  Run tests: `npm test`.

## Testing

We use `vitest` for testing. Ensure all tests pass before submitting a PR.
If you add new features, please add corresponding tests in `test/index.test.ts`.

## Release

Bump the version in `package.json` and follow standard npm publish procedures.
